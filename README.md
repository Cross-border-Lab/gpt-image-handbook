# GPT Image 2 逆向提示词工具

🎨 通过系统化分析，精准复刻任何 GPT Image 2 生成的图片

## 简介

这是一个专业的 AI 图片提示词逆向工程工具，能够通过 8 维度分析框架，将任何图片反推为高质量的提示词，实现 90% 以上的还原度。

## 核心特性

- **8 维度系统分析** - 风格、构图、色彩、光影、主体、环境、技术参数、文字元素
- **双版本输出** - 复杂版（高还原度）+ 简化版（快速上手）
- **90%+ 还原度** - 科学拆解，不是瞎猜
- **结构化输出** - Markdown 层级格式，清晰易读
- **通用兼容** - 支持 GPT Image 2、DALL-E 3、Midjourney、Stable Diffusion
